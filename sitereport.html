Navigation bar

On a webpage, the navigation bar contains a direct link to all the pages. This remains constant throughout the whole website. In this project, the navigation bar is fixed at the top of the device screen. In phone screen, the menu items are hidden in the hamburger menu.

Fig: Navigation bar

The hamburger icon is clickable because input was used to insert a checkbox. The actual icon for the menu items is created using the label. The maximum height of list items is first set to 0, then it is modified such that the menu is visible when the checkbox is selected.

Fig: Hamburger icon

Fig: Hamburger icon

To create the hamburger icon absolute positioning was initially used but then it was fixed. Then, div was used instead of span. While checking code validation, it was concluded div is a block element and it cannot go inside a label. Finally, div was replaced with span.

Home Page

The primary color scheme used on the website is green and yellow. Yellow stands for faith, whereas green is the color of nature and beauty. So, using this color scheme gives the website a more elegant and aesthetically pleasant appearance.

The first page of “index.html” contains an image and a quote. The second section also contains a picture along a small about me section. The first page was first designed on phone then, the CSS for laptop screen was added using media queries.The text that is a little bit lengthier is written in a serif font. Hardly nothing changes on this page throughout the duration of the project. Grid has been used to amke this page responsive.
